Build Daily Task Checklist System (with frequency + staff tracking)

---

1. CREATE TASK TEMPLATES TABLE

---

Table: task_templates

Fields:

* id
* title
* description
* frequency (daily | weekly | monthly | quarterly | half_yearly | yearly)
* assigned_to (staff_id)
* is_active (boolean)
* created_by
* created_at

---

2. CREATE CHECKLIST ITEMS TABLE

---

Table: task_template_items

Fields:

* id
* template_id
* item_text
* order_index

---

3. AUTO GENERATE DAILY TASKS

---

Every day:

Generate tasks from templates based on frequency:

Daily → every day
Weekly → based on selected weekday
Monthly → based on selected date

Create table:

tasks

Fields:

* id
* template_id
* staff_id
* date
* status (pending | completed)
* started_at
* completed_at
* time_taken

---

4. TASK UI (VERY IMPORTANT)

---

For each staff:

Show:

[ ] Call donors
[ ] Update donor records
[ ] Send WhatsApp follow-ups

Each item:

✔ Checkbox
📝 Text beside

---

5. START + COMPLETE FLOW

---

When first checkbox clicked:

→ started_at = now

When all checklist items checked:

→ completed_at = now
→ calculate time_taken

---

6. PARTIAL COMPLETION

---

Allow:

✔ Some items checked
✔ Some unchecked

Status:

* pending
* in_progress
* completed

---

7. FOUNDER CONTROLS

---

Founder can:

✔ Create templates
✔ Edit templates
✔ Delete templates
✔ Add/remove checklist items
✔ Change frequency
✔ Assign staff

---

8. EDIT TEMPLATE UI

---

Allow:

* Add item
  ✏ Edit item
  🗑 Delete item

---

9. STAFF VIEW

---

Staff can:

✔ Only see assigned tasks
✔ Only tick checkboxes
❌ Cannot edit template

---

10. FILTERS

---

Add filters:

* Today
* This week
* This month

---

11. MISSED TASK LINK

---

If:

Task not completed today

→ mark as MISSED

---

12. UI DESIGN

---

Clean layout:

Checkbox + text
Completion progress bar:

Example:

3/5 tasks completed

---

13. OPTIONAL (VERY GOOD)

---

Show:

⏱ Time taken after completion

Example:

✔ Daily report submitted (18 mins)

---

GOAL

Simple daily execution system:

* Staff just tick tasks
* Founder tracks everything
* No confusion
