PART 4 — DONOR LOYALTY BADGE

Calculate loyalty based on years.

Logic:

```
0–1 years   → New Donor
1–3 years   → Active Donor
3–7 years   → Loyal Donor
7+ years    → Champion Donor
```

Add helper function in frontend:

```ts
function getDonorLoyalty(donorSince: string) {
  const years = new Date().getFullYear() - new Date(donorSince).getFullYear()

  if (years >= 7) return "Champion Donor"
  if (years >= 3) return "Loyal Donor"
  if (years >= 1) return "Active Donor"
  return "New Donor"
}
```

Show badge near donor name.
