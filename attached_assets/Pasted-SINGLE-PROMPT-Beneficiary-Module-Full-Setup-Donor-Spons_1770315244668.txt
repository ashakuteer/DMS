SINGLE PROMPT: Beneficiary Module Full Setup + Donor Sponsorship Linking

Build a complete Beneficiaries module for NGO DMS with Homes, Beneficiary Profiles (with photo upload), Updates/Timeline, and Donor ↔ Beneficiary Sponsorship linking.

A) Core Concepts

We have 3 “Homes” (can add more later):

Orphan Girls Home

Visually Challenged Boys Home

Old Age Home

A Beneficiary is an individual living in one home.

Beneficiary can be a child or senior.

Beneficiary has a profile + timeline + updates.

Sponsorship linking:

One Donor can sponsor many Beneficiaries.

One Beneficiary can have many Donors (optional).

Sponsorship can be:

Monthly sponsor

One-time sponsor

Item sponsor (rice, groceries, medicines, stationary, etc.)

Sponsorship must store: start date, optional end date, amount/value, type, notes, status (ACTIVE/PAUSED/ENDED).

Birthday logic:

For Beneficiaries, DOB should be optional.

If DOB is unknown, allow saving only Month & Day (no year).
Store as dobMonth (1–12), dobDay (1–31), and optional dobYear (nullable).

Same approach for special days if needed later.

Updates to Beneficiaries:

Staff can add periodic updates (text + photos).

Updates should show in Beneficiary Timeline.

Updates should be easily copyable for manual WhatsApp/email sending to donors (manual is fine).
