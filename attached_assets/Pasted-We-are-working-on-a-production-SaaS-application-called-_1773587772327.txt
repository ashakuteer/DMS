CURRENT BUG

The donor list page works, but clicking a donor profile causes a crash.

Console errors show:

404: /api/donors/special-occasions
No QueryClient set, use QueryClientProvider

We must fix this.

---

PART 1 — FIX MISSING API

Frontend is calling:

GET /api/donors/:id/special-occasions

But backend does not implement this endpoint.

Add controller:

apps/api/src/modules/donors/donors.controller.ts

```ts
@Get(':id/special-occasions')
async getSpecialOccasions(@Param('id') id: string) {
  return {
    birthdays: [],
    anniversaries: []
  };
}
```

This prevents 404 errors.

---

PART 2 — FIX REACT QUERY PROVIDER

Open:

apps/web/src/app/layout.tsx

Wrap application with QueryClientProvider.

Install if missing:

npm install @tanstack/react-query

Add:

```tsx
import { QueryClient, QueryClientProvider } from '@tanstack/react-query'

const queryClient = new QueryClient()

export default function RootLayout({ children }) {
  return (
    <QueryClientProvider client={queryClient}>
      {children}
    </QueryClientProvider>
  )
}
```

This fixes:

No QueryClient set error.

---
