Build a new Dashboard section called “Analytics” (admin-only) that shows donor + donation + sponsorship insights with charts and actionable lists. Use existing DB tables for donors, donations, pledges, beneficiaries, sponsorships, and reminders. Do not break existing pages.

A) ROUTES / NAV
1) Add sidebar item: “Analytics”
2) Create page: /dashboard/analytics

B) KPI CARDS (TOP ROW)
Show these cards with current values + small trend vs previous period (where possible):
1) Total Donors
2) Total Donations (₹) - This month
3) Total Donations (₹) - Trailing 12 months
4) Donation Count - This month
5) Active Sponsorships (count) + total monthly ₹ (sum)
6) Pledges Pending (count + ₹)
7) Donors With Special Days in Next 30 days (count)

C) CHARTS (MIDDLE)
Use chart library already used (or add Recharts if not present).
Add these charts:
1) Donations Amount by Month (last 12 months) - line/area chart
2) Donation Count by Month (last 12 months) - line/bar
3) Donations by Type (Cash vs In-kind categories) - pie/donut
4) Donations by Home (Girls/Blind Boys/Old Age/General) - bar chart
5) Sponsorships Due vs Paid (last 6 months) - stacked bar (if paid tracking exists; if not, show active sponsorships due this month vs overdue)

D) DONOR SEGMENTS (TABS)
Create tabs with tables + quick actions:
Tab 1: “Top Donors (This FY)”
- top 20 donors by total amount this FY
- show: donor name/code, total ₹, count, last donation date
- action: open donor profile

Tab 2: “Donors At Risk”
Definition: donors who donated before but have not donated in the last N days:
- compute expected interval from their history:
  - if donor has donationFrequency field, use it
  - else infer from last 3 donations median gap
- mark as at-risk if today > (lastDonationDate + expectedGap + graceDays)
Use graceDays = 7
Show: donor, last donation date, expected next window, risk score (Low/Med/High)
Actions:
- Copy WhatsApp reminder (manual)
- Queue email reminder (if email exists)

Tab 3: “Pledges Due”
- list pledges with status PENDING/POSTPONED
- show promised date, amount/value, donor, notes
Actions:
- Mark Fulfilled
- Postpone (choose new date)
- Copy WhatsApp
- Queue Email

Tab 4: “Sponsorships Due (This Month)”
- list active monthly sponsors due in next 7 days + overdue
Show: donor, beneficiary, home, amount, due day, status badge
Actions:
- Mark Paid (if feature exists; else mark “Done for this month”)
- Copy WhatsApp reminder
- Queue Email reminder

Tab 5: “Special Days Next 30 Days”
- list donor birthdays/anniversaries/memorial days in next 30 days
Show: donor, date, type, assigned staff (if any)
Actions:
- Copy WhatsApp wish
- Queue Email wish
- Open donor profile
